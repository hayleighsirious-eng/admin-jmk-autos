SETTINGS & SETUP

Configuration and Supabase setup guide

🔑 HOW TO FIND YOUR SUPABASE URL & KEY

1. Go to supabase.com and log in
2. Click on your project (e.g. jmkautos)
3. In the left sidebar, click Project Settings (gear ⚙️ icon)
4. Click API under the Configuration section
5. You'll see:

Project URL: https://abcdefghijkl.supabase.co ← This is your SUPABASE_URL anon public: eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9... ← This is your SUPABASE_KEY

Then open this file (admin/index.html) and replace the two constants at the top of the <script> section.

🗄️ DATABASE SQL — Run in Supabase SQL Editor

Go to SQL Editor in your Supabase project and run this to add the extra columns:

-- Add extra columns to your existing cars table ALTER TABLE cars ADD COLUMN IF NOT EXISTS price bigint; ALTER TABLE cars ADD COLUMN IF NOT EXISTS price_label text; ALTER TABLE cars ADD COLUMN IF NOT EXISTS category text; ALTER TABLE cars ADD COLUMN IF NOT EXISTS in_stock boolean default true; ALTER TABLE cars ADD COLUMN IF NOT EXISTS torque text; ALTER TABLE cars ADD COLUMN IF NOT EXISTS power_kw text; ALTER TABLE cars ADD COLUMN IF NOT EXISTS dimensions text; ALTER TABLE cars ADD COLUMN IF NOT EXISTS vin text; ALTER TABLE cars ADD COLUMN IF NOT EXISTS location text default 'Ikotun Egbe, Lagos'; -- Enable RLS on cars table (important for security) ALTER TABLE cars ENABLE ROW LEVEL SECURITY; -- Allow public reads (needed for your public site) CREATE POLICY "Public can read available cars" ON cars FOR SELECT USING (true); -- Allow only authenticated users (admin) to insert/update/delete CREATE POLICY "Admin can do everything" ON cars FOR ALL USING (auth.role() = 'authenticated');

🪣 STORAGE BUCKET SETUP

1. In Supabase, go to Storage (left sidebar)
2. Click New bucket
3. Name it exactly: car-images
4. Enable Public bucket toggle → Save
5. Click the bucket → Policies tab → Add policy:
   • Allow authenticated users to upload (INSERT)
   • Allow public to view (SELECT)

👤 ADMIN USER SETUP

1. In Supabase, go to Authentication → Users
2. Click Invite user or Add user
3. Enter your email (e.g. admin@jmkautos.com) and a strong password
4. Use those exact credentials to log in to this admin panel
